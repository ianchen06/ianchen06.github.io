# Causal Analysis

## Why are regression models not enough?

Regression models are great for predicting the value of a variable given a set of other variables. However, they are not good at answering questions about the causal relationship between variables. For example, if we want to know whether smoking causes cancer, we can't just build a regression model to predict cancer given smoking. This is because there are many other factors that can cause cancer, such as genetics, diet, and exercise. If we don't account for these factors, we will get a biased estimate of the effect of smoking on cancer.
